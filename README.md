# Estudo básico de PHP 001

## Estudo e implementação de exemplos e exercícios do livro "PHP Crash Course" de Matt Smith.

## Descrição do livro

> Este material didático, de autoria do Dr. Matt Smith, oferece um roteiro prático e evolutivo sobre o desenvolvimento web moderno com PHP 8.x.
> O conteúdo inicia com os fundamentos da linguagem, demonstrando a transição de scripts simples e manipulação de arquivos para o desenvolvimento de aplicações web estruturadas e dinâmicas. 
> Através da aplicação da arquitetura Model-View-Controller (MVC), o autor guia o leitor na organização de softwares de forma escalável e profissional. 
> São explorados conceitos essenciais para o ecossistema PHP atual, como a gestão de dependências via Composer e o design eficiente de templates com a biblioteca Twig. 
> As fontes detalham profundamente a Programação Orientada a Objetos (POO), abordando herança,interfaces, traits e o tratamento de erros com exceções. 
>Por fim, o guia foca em tópicos críticos como a persistência de dados via PDO e bibliotecas ORM(Doctrine), além de implementar rigorosas práticas de segurança, como o armazenamento de senhas com hashing e a prevenção de ataques de injeção de SQL.
